# atmolyt-host

Основное приложение/прошивка для "хоста" системы Atmolyt.

## Назначение

- Сбор данных с локальных и удалённых датчиков (air/noise/IMU/RTC).
- Отображение текущих значений и базовой статистики.
- Управление конфигурацией системы (пороги, частота опроса, режимы питания).
- Хранение логов и (опционально) передача данных наружу (ПК, сервер и т.п.).

## Статус

Базовый каркас для последующей интеграции модулей сенсоров и UI.
